# Employee-Attrition-Analysis-Dashboard-Power-BI-Project

Understanding employees is just as important as understanding customers. In this project, I built an HR Analytics Dashboard using Power BI to explore key workforce insights such as employee attrition, department distribution, salary patterns, and demographic trends.

The goal of this project was to transform raw HR data into meaningful visual insights that can help organizations make smarter workforce decisions.

🚀 Project Overview

This dashboard provides a clear overview of employee data and highlights important HR metrics such as:

Total Employees

Attrition Count & Attrition Rate

Average Salary

Average Age of Employees

Department-wise Employee Distribution

Attrition by Education, Age Group, and Job Role

With interactive filters and visuals, users can easily explore workforce patterns and identify areas that may need attention.

📌 Key Features

✔ Interactive Power BI dashboard
✔ Employee attrition analysis
✔ Department and job role insights
✔ Age group and education distribution
✔ Easy-to-understand KPI metrics

The dashboard is designed to make HR data simple, visual, and actionable.
